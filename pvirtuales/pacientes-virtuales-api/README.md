# Backend API - Pacientes Virtuales

Este directorio contiene todo el código fuente de la API del proyecto "Pacientes Virtuales".

## Puntos Clave
- **Framework**: PHP nativo siguiendo patrones MVC.
- **Estándar**: PSR-4 para autoloading.
- **Gestor de Dependencias**: Composer.

## Puesta en Marcha
1. Ejecutar `composer install` para instalar las dependencias.
2. Configurar las variables de entorno (ej. base de datos) en la carpeta `config/`.
3. Iniciar el servidor con `php -S localhost:8000 -t public`.
