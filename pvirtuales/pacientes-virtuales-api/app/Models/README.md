# Directorio `Models` (o Entidades)

Este directorio contiene los **Modelos**.

Cada archivo aquí representa una tabla de la base de datos (ej. `User.php` para la tabla `users`). Su función es:
- Definir la estructura de los datos y sus campos.
- Establecer las relaciones entre tablas (ej. un Usuario tiene muchos Pacientes).

Son clases "planas" (POPOs - Plain Old PHP Objects) que actúan como un mapa de los datos.
