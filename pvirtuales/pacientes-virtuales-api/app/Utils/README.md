# Directorio `Utils` (o `Helpers`)

Este directorio contiene clases y funciones de ayuda reutilizables en toda la aplicación.

Son componentes genéricos que no encajan en ninguna otra capa, como:
- Clases para formatear respuestas de la API (`ApiResponse.php`).
- Funciones para manejar fechas o cadenas de texto.
- Componentes para interactuar con APIs externas.
