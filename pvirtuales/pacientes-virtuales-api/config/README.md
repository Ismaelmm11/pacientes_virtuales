# Directorio `config`

Contiene todos los archivos de configuración de la aplicación.

Estos archivos permiten modificar el comportamiento de la aplicación sin cambiar el código fuente. Ideal para gestionar diferentes entornos (desarrollo, producción).

**Ejemplos:**
- `database.php`: Credenciales de la base de datos.
- `app.php`: Nombre de la app, modo debug, zona horaria.
- `services.php`: Claves de APIs externas.
