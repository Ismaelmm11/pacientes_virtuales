# Directorio `assets`

Almacena todos los recursos estáticos que la aplicación necesita.

- **`images/`**: Para imágenes rasterizadas (PNG, JPG).
- **`svgs/`**: Para imágenes vectoriales (SVG).
- **`fonts/`**: Para fuentes de texto personalizadas.
- **`json/`**: Para archivos de configuración o datos locales en formato JSON.
